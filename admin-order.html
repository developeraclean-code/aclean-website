Kelola Layanan

Tambah layanan baru per kategori & sub-kategori. Layanan bawaan tidak bisa dihapus.

1
Buat Google Spreadsheet baru
Buka Google Drive → New → Google Sheets
  • Buka drive.google.com → klik tombol + New
  • Pilih Google Sheets → beri nama: "AClean Orders"
  • Biarkan spreadsheet kosong — Apps Script akan membuat kolom otomatis
2
Buka Apps Script Editor
Di dalam spreadsheet tadi
  • Di Google Sheets, klik menu Extensions → Apps Script
  • Hapus semua kode yang ada (biasanya ada function myFunction(){})
  • Paste kode di bawah ini, lalu klik 💾 Save (Ctrl+S)
// ─── ACLEAN SERVICE — Google Apps Script ──────────────────────────── // Paste ke Apps Script Editor, lalu Deploy sebagai Web App const SHEET_NAME = 'Orders'; const SECRET_TOKEN = 'aclean-secret-2026'; // Ganti sesuai keinginan Anda function doPost(e) { try { const data = JSON.parse(e.postData.contents); // Update status order if (data.action === 'updateStatus') { if (data.token !== SECRET_TOKEN) return deny(); const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME); if (!sheet) return ok({updated: false}); sheet.getRange(data.rowIndex, 17).setValue(data.status); sheet.getRange(data.rowIndex, 18).setValue(new Date().toLocaleString('id-ID')); return ok({updated: true}); } // Simpan settings harga if (data.action === 'saveSettings') { if (data.token !== SECRET_TOKEN) return deny(); const ss = SpreadsheetApp.getActiveSpreadsheet(); let s = ss.getSheetByName('Settings') || ss.insertSheet('Settings'); s.clearContents(); s.appendRow(['Key', 'Value', 'Updated']); s.appendRow(['services', JSON.stringify(data.services), new Date().toISOString()]); s.appendRow(['transportFee', data.transportFee, '']); s.appendRow(['freeTransportMin', data.freeTransportMin, '']); return ok({saved: true}); } // Default: Simpan order baru const sheet = getOrCreateSheet(); const transport = data.qty >= (data.freeTransportMin || 2) ? 0 : (data.transportFee || 20000); const total = (data.price * data.qty) + transport; sheet.appendRow([ new Date().toLocaleString('id-ID'), // A: Timestamp data.orderId, // B: Order ID data.nama, // C: Nama data.wa, // D: WhatsApp data.area, // E: Area data.alamat, // F: Alamat data.koordinat || '', // G: Koordinat GPS data.svcName, // H: Layanan data.ukuran, // I: Ukuran AC data.qty, // J: Jumlah Unit data.tanggal, // K: Tanggal data.waktu, // L: Waktu data.price, // M: Harga/unit transport, // N: Transport total, // O: Total data.catatan || '-', // P: Catatan 'Pending', // Q: Status '' // R: Updated At ]); return ok({orderId: data.orderId, total: total}); } catch(err) { return err_r(err.toString()); } } function doGet(e) { if (e.parameter.token !== SECRET_TOKEN) return deny(); // Ambil settings if (e.parameter.action === 'settings') { const s = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('Settings'); if (!s) return ok({services: null, transportFee: 20000, freeTransportMin: 2}); const rows = s.getDataRange().getValues().slice(1); const cfg = {}; rows.forEach(r => { if(r[0]) cfg[r[0]] = r[1]; }); return ok(cfg); } // Ambil data order const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME); if (!sheet) return ok({orders: []}); const rows = sheet.getDataRange().getValues(); if (rows.length < 2) return ok({orders: []}); const headers = rows[0]; const orders = rows.slice(1).map((row, i) => { const o = {rowIndex: i + 2}; headers.forEach((h, j) => { o[h] = row[j]; }); return o; }); return ok({orders: orders.reverse()}); // terbaru di atas } function getOrCreateSheet() { const ss = SpreadsheetApp.getActiveSpreadsheet(); let s = ss.getSheetByName(SHEET_NAME); if (!s) { s = ss.insertSheet(SHEET_NAME); s.appendRow(['Timestamp','Order ID','Nama','WhatsApp','Area','Alamat', 'Koordinat','Layanan','Ukuran AC','Jumlah Unit','Tanggal', 'Waktu','Harga/Unit','Transport','Total','Catatan','Status','Updated At']); s.setFrozenRows(1); s.getRange('1:1').setBackground('#1B3A7B').setFontColor('#FFFFFF').setFontWeight('bold'); } return s; } function ok(data) { return res(JSON.stringify({success:true,...data})); } function deny() { return res(JSON.stringify({success:false,error:'Unauthorized'})); } function err_r(msg) { return res(JSON.stringify({success:false,error:msg})); } function res(body) { return ContentService.createTextOutput(body).setMimeType(ContentService.MimeType.JSON); }
Ganti SECRET_TOKEN dengan token rahasia pilihan Anda. Token ini juga harus dimasukkan di pengaturan admin panel.
3
Deploy sebagai Web App
Ini menghasilkan URL untuk menerima data order
  • Klik tombol Deploy (kanan atas) → pilih New deployment
  • Klik ikon ⚙️ di samping "Select type" → pilih Web app
  • Description: "AClean Order API"
  • Execute as: Me (developer.aclean@gmail.com)
  • Who has access: Anyone (agar form order bisa mengirim data)
  • Klik Deploy → klik Authorize access → pilih akun Google Anda
  • Kopi Web app URL yang muncul (format: https://script.google.com/macros/s/.../exec)
4
Pasang URL di Admin Panel
Agar order tersimpan ke Google Sheets
  • Buka tab Harga & Transport di sidebar kiri
  • Scroll ke bagian URL Apps Script
  • Paste URL yang tadi dikopi, klik Simpan
  • URL akan otomatis tersimpan dan dipakai di halaman order
Setelah setup selesai, setiap order dari halaman order.html akan otomatis tersimpan ke Google Sheets Anda!
5
(Opsional) Setup Google Sign-In
Untuk login admin dengan akun Google
  • Buka console.cloud.google.com → buat project baru: "AClean"
  • Cari APIs & Services → Credentials → + Create Credentials → OAuth 2.0 Client ID
  • Application type: Web application
  • Authorized JavaScript origins: https://aclean.id
  • Kopi Client ID yang muncul
  • Buka file admin-order.html → cari teks GANTI_DENGAN_GOOGLE_CLIENT_ID_ANDA → ganti dengan Client ID Anda
Sebelum Google Client ID dikonfigurasi, gunakan PIN login di halaman login. Bisa diganti di konfigurasi bawah.
